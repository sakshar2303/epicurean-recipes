# 🍲 Epicurean - Premium Recipe Discovery

Epicurean is a state-of-the-art culinary exploration platform designed for food enthusiasts who appreciate quality and elegance. Discover your next signature dish with our curated collections, trending flavors, and global kitchen.

![Epicurean Hero](https://images.unsplash.com/photo-1543353071-873f17a7a088?ixlib=rb-1.2.1&auto=format&fit=crop&w=1350&q=80)

## ✨ Features

- **Modern UI/UX**: A premium, responsive design built with a focus on aesthetics and user experience.
- **Trending Flavors**: Real-time trending recipes from around the globe.
- **Green Gourmet**: Dedicated section for vegetarian and healthy living enthusiasts.
- **World Kitchen**: Explore cuisines from various cultures with curated categories.
- **Fast Search**: Precision search to find exactly what you're craving.
- **Detailed Insights**: Complete ingredient lists and step-by-step instructions.

## 🛠️ Built With

- **React.js**: Front-end framework.
- **Styled Components**: Robust CSS-in-JS for a modern design system.
- **Framer Motion**: Smooth, premium animations for fluid interactions.
- **Splide.js**: High-performance carousels for recipe browsing.
- **Spoonacular API**: Reliable and extensive recipe data.

## 🚀 Getting Started

### Prerequisites

- Node.js & npm
- Spoonacular API Key

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/epicurean.git
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Create a `.env` file in the root directory and add your API key:
   ```env
   REACT_APP_API_KEY=your_api_key_here
   ```
4. Start the development server
   ```bash
   npm start
   ```
